## Features
-> Add tasks with:
* Title
* Category
*Priority (Low / Medium / High)
*Due date (optional)

-> Edit tasks anytime
-> Mark tasks as completed or uncompleted

-> View:

* All tasks

* Completed tasks

* Pending tasks

-> Search tasks by keyword

-> Sort tasks:

* By priority

* By due date

* Alphabetically

-> Clear completed tasks
-> Delete all tasks (with confirmation)
-> Automatic saving to tasks.json
-> Automatic backup to tasks_backup.json

## Technologies Used

* Python 3 – Core language

* JSON – Stores tasks in tasks.json

* OS module – Checks file existence

* Datetime module – Handles due dates

* Shutil (copyfile) – Creates automatic backups
